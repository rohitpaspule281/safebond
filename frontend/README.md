# Safebond Frontend

Premium Next.js + Tailwind interface for the Safebond mental wellness platform.

## Implemented surfaces

- marketing landing page
- login and signup pages
- authenticated workspace shell
- dashboard overview
- chat workspace
- mood analytics view

## Design direction

- calming sage, sand, and soft coral palette
- editorial typography with `Fraunces` and `Manrope`
- glass-like panels and layered gradients
- startup-style analytics surfaces instead of CRUD tables
- responsive layout with motion-enhanced entry transitions

## Commands

```bash
npm install
npm run dev
```

## Environment

Create `.env.local` from `.env.local.example`:

```bash
cp .env.local.example .env.local
```
